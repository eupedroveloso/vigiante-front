Tools Bar Map

Barra de ferramentas do mapa com sequência fixa: menu (bars), mode-tool (mapa/lista), borda interna, desenhar polígono, desenhar círculo e camadas.

Import


import {{ '{' }} ToolsBarMapComponent {{ '}' }} from '@/shared/components/tools-bar-map/tools-bar-map.component';

@@Component({{ '{' }}
  imports: [ToolsBarMapComponent]
{{ '}' }})

Exemplo

Sequência: bars → mode-tool → border-inner → draw-polygon → draw-circle → layer-group.


<app-tools-bar-map
  [modeToolOptions]="modeOptions"
  [modeToolValue]="modeValue()"
  (menuClick)="onMenuClick()"
  (toolClick)="onToolClick($event)"
  (modeChange)="onModeChange($event)"
/>

Props

Prop Type Default Description
modeToolValue string '' Valor selecionado no mode-tool (ex: 'map', 'list')
modeToolOptions ModeToolOption[] [] Opções do mode-tool (mapa, lista)

Events

Event Payload Description
menuClick void Clique no botão menu (bars)
toolClick string Clique em uma ferramenta: 'border-inner' | 'draw-polygon' | 'draw-circle' | 'layer-group'
modeChange string Mudança de modo no mode-tool (map/list)

Sequência dos botões

  1. bars.svg — Menu
  2. app-mode-tool — Seletor mapa/lista
  3. border-inner.svg — Borda interna
  4. draw-polygon.svg — Desenhar polígono
  5. draw-circle.svg — Desenhar círculo
  6. layer-group.svg — Camadas
