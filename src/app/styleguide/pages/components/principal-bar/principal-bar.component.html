Principal Bar

Barra principal de ações com menu toggle, botões de ação e campo de busca.

Import

      
import {{ '{' }} PrincipalBarComponent {{ '}' }} from '@/shared/components/principal-bar/principal-bar.component';

@@Component({{ '{' }}
  imports: [PrincipalBarComponent]
{{ '}' }})

Design completo (Figma 309-1542)

Três linhas: barra escura (project-tabs), toolbar (menu, mode-tool, ações, input com ícone, notificações), breadcrumb com ícones à direita.

Exemplo Padrão

Barra principal com menu, mode-tool, ações, input de busca (com ícone e divisor) e notificações.

      
<app-principal-bar
  [showMenuToggle]="true"
  [actions]="actions"
  [showSearch]="true"
  [showNotifications]="true"
  [showProjectTabs]="false"
  [showModeTool]="true"
  [modeToolOptions]="modeToolOptions"
  [modeToolValue]="modeToolValue()"
  (menuToggleClick)="onMenuToggle()"
  (actionClick)="onActionClick($event)"
  (searchChange)="onSearchChange($event)"
  (notificationClick)="onNotificationClick()"
  (modeToolChange)="onModeToolChange($event)"
/>

Props

Prop Type Default Description
showMenuToggle boolean true Exibe o botão de toggle do menu
actions PrincipalBarAction[] [] Array de ações/botões a serem exibidos
searchPlaceholder string 'Buscar local' Placeholder do campo de busca
showSearch boolean true Exibe o campo de busca
showNotifications boolean true Exibe o botão de notificações
showModeTool boolean true Exibe o seletor de modo (mode-tool: mapa/lista)
modeToolOptions ModeToolOption[] [] Opções do mode-tool (ex.: mapa, lista)
modeToolValue string '' Id da opção selecionada no mode-tool
showToolsBarMap boolean false Usa o tools-bar-map (menu + mode-tool + ícones: border-inner, draw-polygon, draw-circle, layer-group) em vez de menu + mode-tool + actions
activeToolsBarMapToolId string undefined Id do botão ativo do tools-bar-map (border-inner, draw-polygon, draw-circle, layer-group)
showBreadcrumb boolean true Exibe a barra de breadcrumb abaixo da toolbar
breadcrumbItems BreadcrumbItem[] [] Itens do breadcrumb (label, routerLink?, ellipsis?)
breadcrumbRightActions PrincipalBarBreadcrumbAction[] [] Ícones à direita do breadcrumb (id, icon filename, label?)

Events

Event Payload Description
menuToggleClick void Emitido quando o botão de menu é clicado
actionClick string Emitido quando uma ação é clicada (retorna o id da ação)
searchChange string Emitido quando o valor da busca muda
notificationClick void Emitido quando o botão de notificações é clicado
modeToolChange string Emitido quando o modo no mode-tool muda (retorna o id da opção)
breadcrumbActionClick string Emitido ao clicar em um ícone à direita do breadcrumb (retorna o id da ação)

PrincipalBarAction Interface

        
export interface PrincipalBarAction {{ '{' }}
  id: string;
  icon: string;
  label?: string;
  active?: boolean;
  disabled?: boolean;
{{ '}' }}
Propriedade Type Required Description
id string ✓ Identificador único da ação
icon string ✓ Caminho para o ícone SVG
label string - Label/tooltip da ação
active boolean - Define se a ação está ativa (estilo laranja)
disabled boolean - Desabilita a ação

Accessibility

Keyboard: Todos os botões são focáveis e ativáveis com Enter/Space

ARIA: Labels apropriados aplicados via aria-label e title

Focus: Anel de foco visível usando var(--ring)

Screen Readers: Ícones têm alt text descritivo
