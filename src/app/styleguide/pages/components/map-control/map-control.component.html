Map Control

Painel vertical de controles de mapa (Figma ViconSaga 616-7299): tela cheia, zoom in/out, Street View e miniatura do mapa base.

Import


import {{ '{' }} MapControlComponent {{ '}' }} from '@/shared/components/map-control/map-control.component';

@@Component({{ '{' }}
  imports: [MapControlComponent]
{{ '}' }})

Exemplo

Painel flutuante sobre fundo escuro, como no mapa.


<app-map-control
  (fullscreenClick)="onFullscreenClick()"
  (zoomInClick)="onZoomInClick()"
  (zoomOutClick)="onZoomOutClick()"
  (streetViewClick)="onStreetViewClick()"
  (baseMapThumbnailClick)="onBaseMapThumbnailClick()"
  [baseMapThumbnailUrl]="thumbnailUrl"
/>

Sem miniatura (placeholder)

Quando baseMapThumbnailUrl não é passado, é exibido um placeholder com ícone de mapa.

Props

Prop Type Default Description
showFullscreen boolean true Exibe o botão de tela cheia
showZoom boolean true Exibe os botões de zoom in/out
showStreetView boolean true Exibe o botão Street View (Pegman)
showBaseMapThumbnail boolean true Exibe a área da miniatura do mapa base
baseMapThumbnailUrl string \| undefined undefined URL da imagem da miniatura do mapa base (opcional)

Events

Event Payload Description
fullscreenClick void Clique no botão tela cheia
zoomInClick void Clique em aumentar zoom
zoomOutClick void Clique em diminuir zoom
streetViewClick void Clique no botão Street View
baseMapThumbnailClick void Clique na miniatura do mapa base

Acessibilidade

Agrupamento: role="group" e aria-label="Controles do mapa" no container.

Botões: cada botão possui aria-label descritivo (Tela cheia, Aumentar zoom, Diminuir zoom, Street View, Mapa base).

Teclado: todos os controles são botões focáveis e ativáveis com Enter/Space.
