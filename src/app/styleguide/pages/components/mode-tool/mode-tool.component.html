Mode Tool

Seletor de modo de visualização (ex.: mapa / lista) com ícones e estados ativo/inativo.

Import


import {{ '{' }} ModeToolComponent, ModeToolOption {{ '}' }} from '@/shared/components/mode-tool/mode-tool.component';

@@Component({{ '{' }}
  imports: [ModeToolComponent]
{{ '}' }})

Exemplo padrão

Mapa selecionado e lista selecionado.

Mapa ativo

Modo atual: {{ selectedMode() }}


<app-mode-tool
  [options]="options"
  [(value)]="selectedMode"
  (valueChange)="onModeChange($event)"
/>

Desabilitado

Estado desabilitado.

Props

Prop Type Default Description
options ModeToolOption[] [] Lista de opções (id, icon, label)
value string '' Id da opção selecionada (two-way binding)
disabled boolean false Desabilita o componente

Events

Event Payload Description
valueChange string Id da opção selecionada

ModeToolOption


export interface ModeToolOption {{ '{' }}
  id: string;
  icon: string;
  label?: string;
{{ '}' }}

Acessibilidade

Teclado: Botões focáveis e ativáveis com Enter/Space.

ARIA: role="group", aria-label, aria-pressed no estado selecionado.

Foco: Anel de foco visível (focus-visible:ring).
