Props (app-input)

Prop Type Default Description
label string 'Label' Rótulo acima do campo
placeholder string 'Placeholder' Placeholder do input (text)
description string 'This is an input description.' Texto de ajuda abaixo
iconLeft string undefined Nome do ícone em assets/icons (ex: 'magnifying-glass.svg'); exibe ícone à esquerda e divisor vertical
disabled boolean false Desabilita o campo
error boolean | 'auto' false Estado de erro (borda vermelha); 'auto' com formulário reativo
type string 'text' 'text' ou 'file'
value string '' Valor (use com valueChange para two-way)

Events

Event Payload Description
valueChange string Valor atual do input (type text)

Acessibilidade

Label: label associado ao input via for/id

Foco: anel de foco visível (ring) com var(--ring)

Erro: borda destrutiva (--destructive) para indicar erro

Desabilitado: opacity reduzida e pointer-events-none
