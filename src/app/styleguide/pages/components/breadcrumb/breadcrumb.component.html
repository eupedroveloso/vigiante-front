Prop Type Default Description

ariaLabel string 'breadcrumb' aria-label do container de navegação

BreadcrumbItem


export interface BreadcrumbItem {{ '{' }}
  label?: string;
  routerLink?: string | string[];
  showDropdown?: boolean;
  ellipsis?: boolean;
{{ '}' }}
  • label — texto do item (omitir quando ellipsis for true)
  • routerLink — se omitido, o item é a página atual (último, em negrito)
  • showDropdown — exibe ícone de chevron down ao lado do item
  • ellipsis — renderiza reticências (...) no lugar do item

Acessibilidade

Navegação: role="navigation", aria-label configurável

Página atual: aria-current="page" e aria-disabled="true" no último item

Ellipsis: texto apenas para leitores de tela (sr-only) descrevendo "Mais níveis"

Separador: aria-hidden="true", role="presentation" para não poluir leitura
